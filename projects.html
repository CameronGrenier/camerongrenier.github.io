Portfolio Website

This website is a portfolio website that I made to showcase my work as a developer as I improve my skills throughout my career. It may be a little baren now, but I hope to fill this page with high quality projects that I can be proud of.

I created this site using HTML, CSS, and Javascript, which I had no experience with before starting. I learned these three languages through the development of a number of failed websites, and a lot of youtube.

I made use of bootstrap and a mixture of templates I found on CodePen for the footer of the site. I also made use of the responsive design of the boostrap templates to edit my site for mobile use (which I have not been fully successful at just yet... though I hope to remove the aside in the future).

I made use of Javascript to create the responsive hamburger menu, as well as the website intro video, and the parallax effect throughout the site.

I used Github to do my version control, this was my first time learning this so it may be a little confusing to look at... I realised part way throught that I was commiting to a different repository than my website was hosted on and so I had to transfer to the correct repository. I hope to gain more experience with thus in future work! If you'd like to check out the repository, you can do so here. If you'd like to check out the wrong repository, you can do so here

I hope you enjoy my site, and other projects to come!
